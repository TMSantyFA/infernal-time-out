# Infernal Time Out 🕹️

**Infernal Time Out** es un juego conceptual de lógica y adivinación desarrollado en Godot Engine. El objetivo principal es descubrir el número oculto antes de que se agote el tiempo, guiándote a través de un sistema dinámico de pistas basadas en temperatura (frío/caliente) mientras avanzas por una ambientación temática infernal.

## 🌋 Características principales

* **Mecánica Central:** El jugador introduce números de forma estratégica para descifrar la cifra correcta. El sistema responde con retroalimentación inmediata basada en patrones de temperatura para indicar qué tan cerca o lejos se encuentra del objetivo.
* **Gestión de Tiempo:** Elementos de presión temporal que desafían la velocidad mental del jugador.
* **Estilo Visual:** Arte y estética basada en Pixel Art con una atmósfera infernal oscura y detallada.
* **Tecnología:** Desarrollado íntegramente en Godot Engine utilizando GDScript para la lógica del juego.

## 🎮 Instalación y Cómo jugar

1. Descarga la última versión del ejecutable desde la sección de *Releases* en GitHub.
2. Descomprime el archivo `.zip` en tu computadora.
3. Ejecuta el archivo `InfernalTimeOut.exe` para iniciar la partida.
4. Escribe tus propuestas numéricas en pantalla y analiza las pistas térmicas para ajustar tu siguiente intento antes de que el tiempo expire.
