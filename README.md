# Infernal Time Out 🕹️

Breve descripción de qué trata el juego: 
Un juego de plataformas en 2D desarrollado con Godot Engine
que explora el concepto de adivinar el numero y guiado a su vez por patrones de temperatura.

## Características principales
*   **Mecánicas:** inicia le juego y emiezas a escribir un numero y esperas que el sistema te de una pista de temperatura.
*   **Arte:** Estilo visual (Pixel Art).
*   **Tecnología:** Desarrollado en Godot Engine usando GDScript.

## Instalación / Cómo jugar
1.  Descarga el ejecutable desde GITHUB
2.  Descomprime el archivo.
3.  Ejecuta `InfernalTimeOut.exe`.
